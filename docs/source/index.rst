«I-PS ПОС. КОМПЛЕКТ»
===================================

**Программный комплекс «I-PS ПОС. КОМПЛЕКТ»**, разработанный АО «ИнжПроектСервис», предназначен для создания и хранения данных в программе AutoCAD с последующей возможностью выдачи спецификации в Microsoft Excel. Программный комплекс реализован на языках программирования С# и VBA, включает в себя Windows приложения, плагины для AutoCAD и Excel.

Программный комплекс включает в себя:

1)	«Обновление IPS-ПОC КОМПЛЕКТ» – автоматическая установка последней или выбранной версии плагина для AutoCAD;
2)	«AutoCAD: Земляные работы» – создание оформления стройгенплана на сети, прямая связь плана и профиля, сокращение времени подсчётов объемов, создание ведомостей;
3)	«AutoCAD: Менеджер ссылок» – замена названий файлов документов *.dwg;
4)	«AutoCAD: Ген. план» – расстановка выносок с координатами на выбранных объектах в документе;
5)	«AutoCAD: Дендрология» – автоматическое создание дендроплана и перечётной ведомости. 

Ознакомьтесь с разделом :doc:`Использование` для получения дополнительной информации, в том числе о том, как :doc:`Установить` проект.

.. note::

  Этот проект находится в активной разработке.
